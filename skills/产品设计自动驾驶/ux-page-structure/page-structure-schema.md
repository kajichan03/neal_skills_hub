# Page Structure Schema

> 页面结构文档规范 —— 基于页面类型的结构化定义

---

## 页面类型总览

| 序号 | 页面类型 | 场景枚举 | 核心特征 |
|------|----------|----------|----------|
| 1 | 列表页 | 版本发布单列表、变更记录列表、审批记录列表、漏洞列表 | 表格展示多条记录，支持筛选和操作 |
| 2 | 表单填写页 | 新建工单、新建发布单 | 左侧流程导航 + 右侧表单填写 |
| 3 | 详情页查看态 | 工单详情页、审批详情页、泳道详情页 | Tab 切换详情 + 右侧日志区 |
| 4 | 二次确认弹窗 | 删除二次确认 | 简洁阻断式确认 |
| 5 | 卡片目录页 | 工单卡片目录 | 网格卡片展示，图片+标题+描述 |

---

## 页面类型 1：列表页

### 场景枚举
- 版本发布单列表
- 变更记录列表
- 审批记录列表
- 漏洞列表

### 结构组成

```
┌─────────────────────────────────────────────────────────────────┐
│  标题与操作区                                                    │
│  ─────────────────────────────────────────────────────────────  │
│  页面标题          [筛选器]                        [操作] [操作]   │
│                                                                 │
│  （点击筛选，展开左侧栏；筛选器和操作按钮根据场景配置，不支持的功能隐藏）  │
└─────────────────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────────────────┐
│  列表区                                                          │
│  ─────────────────────────────────────────────────────────────  │
│  ┌────┬─────────┬─────────┬─────────┬─────────┬─────────────┐  │
│  │ ☑  │  表头   │  表头   │  表头   │  表头   │    操作     │  │
│  ├────┼─────────┼─────────┼─────────┼─────────┼─────────────┤  │
│  │ ☑  │  数据   │  数据   │  数据   │  数据   │  [查看][更多]│  │
│  │ ☑  │  数据   │  数据   │  数据   │  数据   │  [查看][更多]│  │
│  │ ☑  │  数据   │  数据   │  数据   │  数据   │  [查看][更多]│  │
│  └────┴─────────┴─────────┴─────────┴─────────┴─────────────┘  │
│                                                                 │
│  （多选框、操作列根据场景配置，不支持的功能隐藏）                │
└─────────────────────────────────────────────────────────────────┘
```

### 区域说明

#### 标题与操作区
| 元素 | 说明 | 可隐藏 |
|------|------|--------|
| 页面标题 | 当前页面名称 | 否 |
| 筛选器 | 下拉筛选条件 | 是（无筛选需求时） |
| 操作按钮 | 新建、导出等 | 是（无操作时） |

#### 列表区
| 元素 | 说明 | 可隐藏 |
|------|------|--------|
| 多选框 | 全选/单选行 | 是（无批量操作时） |
| 表头 | 列标题，可排序 | 否 |
| 数据 | 单元格内容 | 否 |
| 操作 | 行内操作按钮 | 是（无行操作时） |

### YAML 结构

```yaml
page:
  name: "版本发布单列表"
  type: "list"
  
  header:
    title: "版本发布单"
    filters:
      - field: "status"
        label: "状态"
        options: ["全部", "待发布", "已发布", "已取消"]
      - field: "date_range"
        label: "发布日期"
    actions:
      - label: "新建发布单"
        type: "primary"
      - label: "导出"
        type: "default"
  
  list:
    row_selection: true        # 是否支持多选
    columns:
      - field: "id"
        label: "编号"
        sortable: true
      - field: "title"
        label: "标题"
      - field: "status"
        label: "状态"
        tag: true
      - field: "publish_date"
        label: "发布日期"
        sortable: true
    row_actions:
      - label: "查看"
        action: "navigate_detail"
      - label: "编辑"
        action: "open_edit"
        visible: "status == '待发布'"
```

## 页面类型 2：表单填写页

### 场景枚举
- 新建工单
- 新建发布单

### 结构组成

```
┌─────────────────────────────────────────────────────────────────┐
│  标题区                                                          │
│  ─────────────────────────────────────────────────────────────  │
│  新建工单                                                       │
└─────────────────────────────────────────────────────────────────┘
┌───────────────────┬─────────────────────────────────────────────┐
│                   │                                             │
│  流程导航          │              表单区                         │
│  ─────────────    │  ─────────────────────────────────────────  │
│                   │                                             │
│  ● 基本信息       │  ┌──────────────────────────────────────┐   │
│    填写工单基础    │  │ Section：基本信息                     │   │
│    信息           │  │ ───────────────────────────────────── │   │
│                   │  │                                       │   │
│  ○ 详细描述       │  │  * 工单标题                           │   │
│    描述问题详情    │  │  [                              ]     │   │
│                   │  │                                       │   │
│  ○ 附件上传       │  │  * 工单类型      优先级               │   │
│    添加相关文件    │  │  [缺陷  ▼]      [中    ▼]             │   │
│                   │  │                                       │   │
│                   │  │  Section：详细描述                    │   │
│                   │  │  ─────────────────────────────────────│   │
│                   │  │  [富文本编辑区域                    ] │   │
│                   │  │  [                                ]   │   │
│                   │  └──────────────────────────────────────┘   │
│                   │                                             │
└───────────────────┴─────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────────────────┐
│  操作按钮（吸底）                                                │
│  ─────────────────────────────────────────────────────────────  │
│                         [上一步] [下一步]                       │
│                                         （最后一步显示[确定]）  │
└─────────────────────────────────────────────────────────────────┘
```

### 区域说明

#### 标题区
| 元素 | 说明 | 可隐藏 |
|------|------|--------|
| 页面标题 | 表单名称，如"新建工单" | 否 |

#### 流程导航（左侧）
| 元素 | 说明 | 可隐藏 |
|------|------|--------|
| 流程步骤列表 | 显示所有步骤，当前步骤高亮 | 是（单步表单时隐藏整个导航） |
| 步骤名称 | 步骤标题 | 否 |
| 步骤描述 | 简短说明该步骤内容 | 可选 |

#### 表单区（中间）
| 元素 | 说明 | 可隐藏 |
|------|------|--------|
| Section 分组 | 按主题分组表单项 | 是（字段少时可平铺） |
| 表单项 | 输入框、下拉框、富文本等 | 否 |
| 必填标记 | * 表示必填 | 否 |

#### 操作按钮（吸底）
| 元素 | 说明 | 显示逻辑 |
|------|------|----------|
| 上一步 | 返回上一流程步骤 | 非第一步时显示 |
| 下一步 | 进入下一流程步骤 | 非最后一步时显示 |
| 确定 | 提交表单 | 最后一步时显示 |

### YAML 结构

```yaml
page:
  name: "新建工单"
  type: "form"
  
  header:
    title: "新建工单"
  
  navigation:
    current: 0                    # 当前步骤索引
    steps:
      - name: "基本信息"
        description: "填写工单基础信息"
      - name: "详细描述"
        description: "描述问题详情"
      - name: "附件上传"
        description: "添加相关文件"
  
  form:
    sections:
      - name: "基本信息"
        fields:
          - field: "title"
            label: "工单标题"
            type: "input"
            required: true
          - field: "type"
            label: "工单类型"
            type: "select"
            options: ["缺陷", "需求", "任务"]
            required: true
          - field: "priority"
            label: "优先级"
            type: "select"
            options: ["紧急", "高", "中", "低"]
            required: true
      
      - name: "详细描述"
        fields:
          - field: "description"
            label: "问题描述"
            type: "rich-text"
            required: true
  
  actions:
    - label: "上一步"
      type: "default"
      visible: "current_step > 0"
    - label: "下一步"
      type: "primary"
      visible: "current_step < total_steps - 1"
    - label: "确定"
      type: "primary"
      visible: "current_step == total_steps - 1"
```

## 页面类型 3：详情页查看态

### 场景枚举
- 工单详情页
- 审批详情页
- 泳道详情页

### 结构组成

```
┌─────────────────────────────────────────────────────────────────┐
│  标题与操作区                                                    │
│  ─────────────────────────────────────────────────────────────  │
│  工单详情                    🔴 待处理        [编辑] [更多▼]   │
└─────────────────────────────────────────────────────────────────┘
┌───────────────────────────────────┬─────────────────────────────┐
│                                   │                             │
│  详情区域                          │      日志或流程区域          │
│  ─────────────────────────────    │      ─────────────────      │
│                                   │                             │
│  [Tab1] [Tab2] [Tab3] [Tab4]      │      ┌────────────────┐     │
│  ─────────────────────────────    │      │ 2024-01-15     │     │
│                                   │      │ 10:30 张三     │     │
│  当前 Tab 内容：                   │      │ 创建工单       │     │
│  ┌────────────────────────────┐   │      ├────────────────┤     │
│  │ Section: 基本信息          │   │      │ 2024-01-15     │     │
│  │ ─────────────────────────  │   │      │ 14:20 李四     │     │
│  │ 类型：缺陷                  │   │      │ 开始处理       │     │
│  │ 优先级：紧急                │   │      ├────────────────┤     │
│  │ 创建人：张三                │   │      │ 2024-01-16     │     │
│  │                            │   │      │ 09:00 李四     │     │
│  │ Section: 问题描述          │   │      │ 提交验收       │     │
│  │ ─────────────────────────  │   │      └────────────────┘     │
│  │ 用户反馈登录时偶尔报错...   │   │                             │
│  └────────────────────────────┘   │                             │
│                                   │                             │
└───────────────────────────────────┴─────────────────────────────┘
```

### 区域说明

#### 标题与操作区
| 元素 | 说明 | 可隐藏 |
|------|------|--------|
| 页面标题 | 详情页名称，如"工单详情" | 否 |
| 状态标签 | 当前记录状态 | 是（无状态字段时） |
| 操作按钮 | 编辑、更多操作等 | 是（无操作时） |

#### 详情区域（左侧）
| 元素 | 说明 | 可隐藏 |
|------|------|--------|
| Tab 导航 | 多个 Tab 切换 | 是（单 Tab 时隐藏） |
| Tab 内容 | 当前选中 Tab 的内容 | 否 |
| Section 分组 | 字段按主题分组 | 可选 |
| 字段 | 标签 + 值 | 否 |

#### 日志或流程区域（右侧）
| 元素 | 说明 | 可隐藏 |
|------|------|--------|
| 时间轴 | 按时间倒序的记录 | 是（无日志时） |
| 流程跟踪 | 审批流程节点状态 | 二选一 |

### YAML 结构

```yaml
page:
  name: "工单详情页"
  type: "detail"
  
  header:
    title: "工单详情"
    status:
      field: "status"
      color_mapping:
        待处理: "warning"
        处理中: "processing"
        已关闭: "success"
    actions:
      - label: "编辑"
        action: "open_edit"
        visible: "status != '已关闭'"
      - label: "更多"
        type: "dropdown"
        items:
          - label: "指派"
          - label: "关闭"
          - label: "删除"
  
  content:
    tabs:
      - name: "基本信息"
        sections:
          - name: "基础字段"
            fields:
              - field: "type"
                label: "类型"
              - field: "priority"
                label: "优先级"
              - field: "reporter"
                label: "创建人"
          - name: "问题描述"
            fields:
              - field: "description"
                label: ""
                type: "rich-text"
      
      - name: "处理记录"
        # 此 Tab 内容与右侧日志区合并展示，或替代右侧区域
      
      - name: "关联信息"
        sections:
          - name: "关联工单"
            type: "link-list"
          - name: "关联代码"
            type: "link-list"
  
  sidebar:
    type: "timeline"              # timeline | flow
    title: "处理记录"
    data_source: "activities"
    fields:
      - field: "timestamp"
        label: "时间"
      - field: "operator"
        label: "操作人"
      - field: "action"
        label: "操作"
      - field: "comment"
        label: "备注"
```

## 页面类型 4：二次确认弹窗

### 场景枚举
- 删除二次确认
- 状态变更确认
- 批量操作确认

### 结构组成

```
┌─────────────────────────┐
│                         │
│       确认删除          │  ← 弹窗标题
│                         │
│  确定要删除该工单吗？    │  ← 内容
│  删除后不可恢复。        │
│                         │
│    [取消]  [确定]       │  ← 按钮
│                         │
└─────────────────────────┘
```

### 区域说明

| 元素 | 说明 | 必含 |
|------|------|------|
| 弹窗标题 | 确认操作的标题 | 是 |
| 内容 | 操作描述和后果提示 | 是 |
| 按钮 | 取消 + 确定（或自定义） | 是 |

### YAML 结构

```yaml
page:
  name: "删除确认"
  type: "confirm"
  
  title: "确认删除"
  content: "确定要删除该工单吗？删除后不可恢复。"
  
  buttons:
    - label: "取消"
      type: "default"
      action: "close"
    - label: "确定"
      type: "primary"
      danger: true              # 危险操作标红
      action: "confirm_delete"
```

## 页面类型 5：卡片目录页

### 场景枚举
- 工单卡片目录
- 应用卡片目录
- 资源卡片目录

### 结构组成

```
┌─────────────────────────────────────────────────────────────────┐
│  标题与操作区                                                    │
│  ─────────────────────────────────────────────────────────────  │
│  工单卡片目录              [分类 ▼] [状态 ▼]    [🔍 搜索...]   │
│                                                                 │
│  （筛选器和搜索根据场景配置，不支持的功能隐藏）                  │
└─────────────────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────────────────┐
│  卡片区域                                                        │
│  ─────────────────────────────────────────────────────────────  │
│                                                                 │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐             │
│  │             │  │             │  │             │             │
│  │   [图片]    │  │   [图片]    │  │   [图片]    │             │
│  │             │  │             │  │             │             │
│  ├─────────────┤  ├─────────────┤  ├─────────────┤             │
│  │ 卡片标题     │  │ 卡片标题     │  │ 卡片标题     │             │

│  └─────────────┘  └─────────────┘  └─────────────┘             │

│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐             │

│  ├─────────────┤  ├─────────────┤  ├─────────────┤             │
│  │ 卡片标题     │  │ 卡片标题     │  │ 卡片标题     │             │
│  │ 卡片描述...  │  │ 卡片描述...  │  │ 卡片描述...  │             │
│  └─────────────┘  └─────────────┘  └─────────────┘             │
│                                                                 │
│  （每个卡片：图片 + 标题 + 描述）                                │
└─────────────────────────────────────────────────────────────────┘
```

### 区域说明

#### 标题与操作区
| 元素 | 说明 | 可隐藏 |
|------|------|--------|
| 页面标题 | 目录名称，如"工单卡片目录" | 否 |
| 筛选器 | 下拉筛选条件（分类、状态等） | 是（无筛选需求时） |
| 搜索框 | 关键词搜索 | 是（无搜索需求时） |

#### 卡片区域
| 元素 | 说明 | 必含 |
|------|------|------|
| 卡片网格 | 多列网格布局 | 是 |
| 卡片图片 | 封面图/图标 | 是 |
| 卡片标题 | 名称 | 是 |
| 卡片描述 | 简短描述 | 是 |

### 卡片规则
- 图片：封面图或默认图标
- 标题：卡片名称
- 描述：一句话简介

### YAML 结构

```yaml
page:
  name: "工单卡片目录"
  type: "card-catalog"
  
  header:
    title: "工单卡片目录"
    filters:
      - field: "category"
        label: "分类"
        options: ["全部", "缺陷", "需求", "任务"]
      - field: "status"
        label: "状态"
        options: ["全部", "待处理", "处理中", "已关闭"]
    search:
      placeholder: "搜索工单..."
      search_fields: ["title", "id"]
  
  cards:
    layout: "grid"                # grid | list
    columns: 3                    # 网格列数（响应式：desktop 3, tablet 2, mobile 1）
    card_template:
      image:
        field: "cover_image"
        fallback: "default-icon"  # 默认图
      title:
        field: "title"
        max_length: 20            # 超长截断
      description:
        field: "summary"
        max_length: 50
      link:
        action: "navigate_detail"
        target: "/tickets/{id}"
    
    pagination:
      type: "load-more"           # load-more | pagination | infinite-scroll
      page_size: 9
```

## 快速选择指南

| 你的场景 | 推荐页面类型 |
|----------|--------------|
| 展示多条记录，需要筛选和排序 | **列表页** |
| 创建或编辑复杂数据，分步填写 | **表单填写页** |
| 查看单条记录的完整信息和历史 | **详情页查看态** |
| 需要用户确认重要操作 | **二次确认弹窗** |
| 以浏览为主，卡片式展示 | **卡片目录页** |
